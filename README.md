# lorenz-system
Interactive and optimised renderer of numerical solution of Lorenz system of equations

![alt text](https://i.ibb.co/rH8rTW7/screenshot.png)

Controls:
1) Use mouse and scroll wheel to orient perspective in the 3D environment
2) Use Q/E to control rho, A/S to control sigma and Z/X to control beta
3) Press P to pause/resume simulation and R to reset
4) Press C to restore camera view to normal
5) Use Left/Right arrow keys to adjust generation speed

Executible located in application.windows folder. Requires java to run. Download from www.java.com/download
